This is my first hands-on manual and automation testing project. While it's a basic implementation, it helped me understand the fundamentals of Selenium, Pytest, and test automation using Python.

## Features

* Manual test cases documented in Excel
* Automated login workflow on SauceDemo
* Product sorting validation (Low → High)
* Price verification using assertions
* Pytest fixtures for browser setup

## Tech Stack

* Python
* Selenium WebDriver
* Pytest
* Git & GitHub

## Learning

Through this project, I learned:

* Selenium WebDriver basics
* Writing automated test cases with Pytest
* Using assertions and explicit waits
* Organizing automation projects
* Version control with Git and GitHub

This project marks the beginning of my automation testing journey, and I'm looking forward to building more advanced testing frameworks and improving my automation skills.
